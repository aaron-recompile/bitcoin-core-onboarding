## 作者简介（公开资料）

Amiti Uttarwar is a Bitcoin Core contributor focusing on P2P, mempool policy, and privacy-related improvements.  
She was one of the first female engineers publicly recognized for direct Bitcoin Core contributions.

她也是最早一批因直接为 Bitcoin Core 做出贡献而被公开认可的女性工程师之一。

## Summary of Original Content  
## 原文内容概览

This article describes the author's personal onboarding experience into Bitcoin Core development, including:

- Foundational background needed to begin contributing  
- How to choose what to focus on  
- How to navigate the Bitcoin Core codebase  
- How to find your first pull request  
- How to use the community, IRC, and online materials  
- Concrete learning resources and developer tools  
- Reflections on open-source collaboration and long-term growth  

本文记录了作者进入 Bitcoin Core 开发世界的亲身经历，主要包括：

- 开始做贡献前需要具备的基础认知  
- 如何选择自己的关注点与学习方向  
- 如何熟悉并穿梭于 Bitcoin Core 代码库  
- 如何寻找并完成自己的第一个 PR  
- 如何利用社区、IRC 与线上资料进行学习  
- 具体的学习资源与开发工具推荐

## Why This Article Matters  
## 这篇文章的重要性

This article has become one of the **canonical onboarding resources** for new Bitcoin Core contributors because it:

- Clearly explains the mindset required for protocol development  
- Documents practical steps for getting started  
- Highlights the culture and expectations within the Core project  
- Contains one of the best curated lists of tools & learning paths  
- Shows what “realistic” onboarding looks like, including confusion, uncertainty, and iteration  

这篇文章已经成为 Bitcoin Core 新人最重要的**经典入门资源之一**，原因包括：

- 清晰阐述了做协议开发所需要的心态与思维方式  
- 系统记录了从零开始参与项目的具体步骤  
- 展示了 Core 项目的文化与对贡献者的期望  
- 提供了一份质量极高的工具与学习路径清单  
- 呈现了“真实的 onboarding 过程”，包括困惑、不确定性和不断试错
